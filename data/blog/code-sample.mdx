# 🥘 마늘 볶음밥

30초면 끝나는 초간단 볶음밥 레시피입니다. 복잡한 과정은 모두 빼고, 마늘의 풍미와 밥알의 식감에만 집중했습니다.

<YouTubeShorts videoId="Dr51Weca3i4" />

## 📋 재료 (Ingredients)

- **밥 1공기** (찬밥 권장)
- **마늘 3쪽, 대파 1대**
- **계란 1개**
- **간장 1스푼, 참기름 1스푼, 식용유**

---

## 👨‍Chef's Log (조리 순서)

1. **준비:** 마늘은 다지고 대파는 썰어둡니다. 계란은 가볍게 풀어주세요.
2. **향 내기:** 팬에 기름을 두르고 다진 마늘을 노릇하게 볶아 마늘 향을 뽑습니다.
3. **스크램블:** 마늘을 한쪽으로 밀고 계란을 부어 빠르게 익힙니다.
4. **마무리:** 밥과 간장을 넣고 센 불에서 수분을 날리듯 볶은 뒤, 대파와 참기름을 섞어 완성합니다.
